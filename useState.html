useState — explication simple

Idée : useState permet de garder une information dans le composant. Exemple concret (sans compteur) : garder en mémoire le texte d'un formulaire.

Exemple React (snippet)

Copie-colle ce code dans un composant React (fichier .jsx) :

// ExempleUseState.jsx
import React, { useState } from "react";

export default function ExempleUseState() {
  // name = valeur actuelle (chaque frappe)
  // setName = fonction pour modifier la valeur
  const [name, setName] = useState("");

  return (
    <div>
      <h2>Ton nom</h2>
      <input
        value={name}
        onChange={(e) => setName(e.target.value)}
        placeholder="Écris ton nom"
      />
      <p>Bonjour {name} </p>
    </div>
  );
}

Logique : le state name retient le texte et le réaffiche à chaque modification.
