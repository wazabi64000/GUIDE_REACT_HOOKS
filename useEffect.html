useEffect — explication simple

Idée : useEffect sert à faire quelque chose quand une valeur change (ou au premier rendu). Exemple concret : sauvegarder automatiquement le texte d'un formulaire dans localStorage.

Exemple React (snippet)

// ExempleUseEffect.jsx
import React, { useState, useEffect } from "react";

export default function ExempleUseEffect() {
  const [note, setNote] = useState("");

  // Chaque fois que 'note' change, on sauvegarde dans localStorage
  useEffect(() => {
    localStorage.setItem("quickNote", note);
  }, [note]);

  return (
    <div>
      <h2>Note rapide</h2>
      <textarea value={note} onChange={(e) => setNote(e.target.value)} />
      <p>La note est sauvegardée automatiquement dans ton navigateur.</p>
    </div>
  );
}

Logique : useEffect observe la variable note et exécute la sauvegarde chaque fois qu'elle change.

Mini-démo (sauvegarde automatique)

Tape du texte et recharge la page : le texte est restauré depuis localStorage.

✅ Ce que tu dois retenir : useEffect = exécuter une action quand une donnée change (ici la sauvegarde automatique).
